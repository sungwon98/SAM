# 그러면 최근 10개(1페이지에 10개 웹툰 게시되어 있어서.) 평점 평균은?
total_rates = 0

cartoons = soup.find_all("div", attrs={"class":"rating_type"})
for cartoon in cartoons :
    rate = cartoon.find("strong").get_text() # 첫 "strong" element 가져온다.
    print(rate)
    total_rates += float(rate) # 가져올 때, 평점들이 str 형태이므로 실수화하고 계산.
print("전체 점수 : ", total_rates)
print("평균 점수 : ", total_rates / len(cartoons))
